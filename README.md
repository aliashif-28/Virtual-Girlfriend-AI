# 💖 Fizza: Your Personal AI Companion

Fizza is a smart, caring, and interactive Virtual AI Assistant built using **Python**, **Google Gemini AI**, and **Hinglish** Natural Language Processing. It is designed to act as a supportive companion that can talk, listen, and assist with real-time information.

## ✨ Key Features
* **🗣️ Bilingual Interaction**: Communicates fluently in Hinglish (Hindi + English) with a respectful and caring tone.
* **🧠 Brain Powered by Gemini**: Uses Google's Gemini 1.5 Flash for high-speed and intelligent responses.
* **🎙️ Voice Navigation**: Fully hands-free interaction using Speech Recognition and Google TTS (Indian Accent).
* **🎥 Dynamic Avatar**: Features a visual interface with "Listen" and "Speak" video states for a realistic feel.
* **☁️ Live Weather**: Provides real-time weather updates for Rajkot (or any city).
* **📅 Date & Time**: Keeps you updated with the current Indian date and time.
* **❤️ Caring Mode**: Reminds you to stay hydrated and take care of your health based on local weather conditions.

## 🛠️ Tech Stack
* **Language**: Python 3.11+
* **AI**: Google Generative AI (Gemini API)
* **GUI**: Tkinter (Python's standard GUI library)
* **Audio**: Pygame & gTTS
* **Computer Vision**: OpenCV (for video stream management)
